在前面，我们学习了栈和队列，有了线性表的基础，栈和队列理解起来也很简单，总结来说就是**限定元素操作位置只能在表两端进行的线性表**

下面是二者进行比较：

| 操作/数据结构 | 栈（Stack）      | 队列（Queue）        |
| ------- | ------------- | ---------------- |
| 实现方式    | 数组，链表         | 数组，链表            |
| 操作原则    | `LIFO`        | `FIFO`           |
| 插入操作    | 只能在栈顶插入（Push） | 只能在队尾插入（EnQueue） |
| 删除操作    | 只能在栈顶删除（Pop）  | 只能在队头删除（DeQueue） |
| 应用场景    | 函数调用，表达式求值    | 缓冲区，任务调度         |
